## 一、Redis 五种数据类型对比

| 类型 | 结构 | 特点 | 适用场景 |
|------|------|------|----------|
| **String** | 键值对 | 最简单，支持数字操作 | 缓存、Session、计数器 |
| **Hash** | field-value | 适合存对象 | 用户信息、购物车 |
| **List** | 双向链表 | 有序，可重复 | 消息队列、最新列表 |
| **Set** | 无序集合 | 无重复，支持交并差 | 点赞、标签、共同好友 |
| **Sorted Set** | 有序集合 | 按分数排序 | 排行榜、延迟队列 |

---

## 二、Redis 命令速查表

| 数据类型 | 常用命令 |
|----------|----------|
| String | SET, GET, INCR, DECR, SETEX, SETNX |
| Hash | HSET, HGET, HGETALL, HMSET, HINCRBY |
| List | LPUSH, RPUSH, LPOP, RPOP, LRANGE, LLEN |
| Set | SADD, SREM, SMEMBERS, SISMEMBER, SCARD, SINTER |
| Sorted Set | ZADD, ZRANK, ZREVRANK, ZSCORE, ZREVRANGE, ZINCRBY |

2. Set：点赞功能
java
@Service
public class LikeService {
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    private static final String LIKE_KEY_PREFIX = "article:like:";
    
    public boolean like(Integer articleId, Integer userId) {
        String key = LIKE_KEY_PREFIX + articleId;
        return redisTemplate.opsForSet().add(key, userId.toString()) > 0;
    }
    
    public Long likeCount(Integer articleId) {
        String key = LIKE_KEY_PREFIX + articleId;
        return redisTemplate.opsForSet().size(key);
    }
    
    public Boolean isLiked(Integer articleId, Integer userId) {
        String key = LIKE_KEY_PREFIX + articleId;
        return redisTemplate.opsForSet().isMember(key, userId.toString());
    }
}
3. Sorted Set：排行榜
java
@Service
public class RankService {
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    private static final String RANK_KEY = "article:rank";
    
    public void updateScore(Integer articleId, double increment) {
        redisTemplate.opsForZSet().incrementScore(RANK_KEY, articleId.toString(), increment);
    }
    
    public List<RankDTO> topN(int n) {
        Set<ZSetOperations.TypedTuple<String>> resultSet =
            redisTemplate.opsForZSet().reverseRangeWithScores(RANK_KEY, 0, n - 1);
        
        List<RankDTO> list = new ArrayList<>();
        for (ZSetOperations.TypedTuple<String> tuple : resultSet) {
            list.add(new RankDTO(tuple.getValue(), tuple.getScore()));
        }
        return list;
    }
}
