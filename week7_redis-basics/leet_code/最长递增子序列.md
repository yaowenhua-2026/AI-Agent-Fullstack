题目描述
找到最长严格递增子序列的长度。

代码
java
class Solution {
    public int lengthOfLIS(int[] nums) {
        int n = nums.length;
        int[] dp = new int[n];
        int maxLength = 0;
        
        for (int i = 0; i < n; i++) {
            dp[i] = 1;  // 至少为 1
            for (int j = 0; j < i; j++) {
                if (nums[j] < nums[i]) {
                    dp[i] = Math.max(dp[i], dp[j] + 1);
                }
            }
            maxLength = Math.max(maxLength, dp[i]);
        }
        return maxLength;
    }
}
核心思路
公式	说明
dp[i] = max(dp[j] + 1)	遍历所有 j < i，找递增序列
