# 线程池学习总结

## 一、七大参数

| 参数 | 作用 |
|------|------|
| corePoolSize | 核心线程 |
| maximumPoolSize | 最大线程 |
| keepAliveTime | 空闲时间 |
| unit | 时间单位 |
| workQueue | 队列 |
| threadFactory | 工厂 |
| handler | 拒绝策略 |

## 二、执行流程
任务 → 核心满？→ 队列满？→ 非核心满？→ 拒绝策略

## 三、容量公式
总容量 = maximumPoolSize + 队列容量

## 四、拒绝策略
- AbortPolicy
- CallerRunsPolicy
- DiscardPolicy
- DiscardOldestPolicy

## 五、Executors 风险

| 方法 | 风险 |
|------|------|
| newFixedThreadPool | OOM |
| newCachedThreadPool | OOM |
| newSingleThreadExecutor | OOM |

👉 推荐：手动创建 ThreadPoolExecutor
